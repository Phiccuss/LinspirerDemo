
# Linspirer Demo 

> 比领创更懂你的mdm
> 
> <p align="right">——ljlVink</p>


针对各种教育平台(联想和华为)**领创**平板的研究


支持系统:API>=27(Android 8.1)

测试过的系统:联想 android8.1 android9 android10

**老系统(安卓8(包括)以下)将不会支持 因为md风格**

华为:emui8-鸿蒙OS 2(emui12) **不支持华为管控机，因为没有华为签名**

出于某些原因 不开源核心代码
